Midnight Purple SA is an independent fork of [Multi Theft Auto: San Andreas](https://multitheftauto.com/) that layers the **FMOD Core API** on top of the existing BASS audio stack. The project exists to bring studio-grade spatial audio — full 3D positioning, environmental reverb, per-channel DSP, and realistic vehicle engine simulation — into MTA:SA servers without breaking a single existing script. The core design principle is simple: **BASS remains completely untouched**. Every sound, stream, and effect written against the standard MTA audio API continues to work exactly as before; FMOD is an entirely additive layer exposed through a new set of `fmod*` Lua globals.

## What's New

### FMOD Audio Engine

The heart of Midnight Purple SA is `CFMODManager`, a C++ singleton that wraps the FMOD Core API and drives the entire audio subsystem:

- **Frame-accurate updates** — `CFMODManager` is ticked every game frame, keeping 3D listener position, velocity, and orientation in sync with the local player camera.
- **Right-handed coordinate system** — The FMOD world is configured in right-handed mode with axes remapped to match GTA:SA's coordinate space, so positions passed from Lua land exactly where you expect them in the world.
- **Three-tier channel hierarchy** — A single master `ChannelGroup` owns three sub-groups — **SFX**, **Ambient**, and **Music** — giving server-side scripts independent volume control over each category without affecting the others.
- **Automatic cleanup and resource tracking** — Every sound handle is associated with the resource that created it. When a resource stops, all of its FMOD channels are released automatically; there are no leaks and no manual teardown required.
- **Pop-free 3D sound start** — Sounds are loaded and created in a *paused* state. All 3D attributes (position, velocity, cone) are applied before the channel is unpaused, eliminating the one-frame positional pop that occurs when attributes are set after playback has already begun.

The upstream MTA:SA codebase is synced to **2026-05-16**. The fork is distributed under the version label **Midnight Purple 1.7**. Release notes and changelog entries for each version are tracked in the repository.
