Midnight Purple SA is an independent fork of [Multi Theft Auto: San Andreas](https://multitheftauto.com/) that layers the **FMOD Core API** on top of the existing BASS audio stack. The project exists to bring studio-grade spatial audio — full 3D positioning, environmental reverb, per-channel DSP, and realistic vehicle engine simulation — into MTA:SA servers without breaking a single existing script. The core design principle is straightforward: **BASS remains completely untouched**. Every sound, stream, and effect written against the standard MTA audio API continues to work exactly as before; FMOD is an entirely additive layer exposed through a new set of `fmod*` Lua globals.

## What's New

### FMOD Audio Engine

Midnight Purple SA replaces the default audio experience with a fully integrated FMOD Core layer running alongside the existing BASS stack:

- **Frame-accurate 3D updates** — Listener position, velocity, and orientation are kept in sync with the local player camera every game frame, so 3D sounds always feel anchored to the world.
- **Matched coordinate space** — The FMOD world is configured to match GTA:SA's coordinate system, meaning positions passed from Lua land exactly where you expect them in-game.
- **Three independent volume categories** — Server-side scripts can control SFX, Ambient, and Music volume independently, without affecting other categories.
- **Automatic resource cleanup** — Every sound handle is tied to the resource that created it. When a resource stops, all of its FMOD audio is released automatically with no manual teardown needed.
- **Pop-free 3D sound start** — Sounds are prepared in a paused state with all 3D attributes applied before playback begins, eliminating the positional pop that can occur with naive implementations.
