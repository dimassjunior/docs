The Midnight Purple SA server component can be built natively on Linux using the included build script. The Linux build targets the **dedicated server only** — the game client (including all FMOD audio features) runs exclusively on Windows.

## Build

Use the `linux-build.sh` script from the repository root to compile the server:

```bash
./linux-build.sh [--arch=x86|x64|arm|arm64] [--config=debug|release] [--cores=<n>]
```

### Flags

| Flag | Description | Default |
|---|---|---|
| `--arch` | Target CPU architecture | `x64` |
| `--config` | Build configuration — `debug` or `release` | `release` |
| `--cores` | Number of parallel compile jobs | auto-detect |

### Examples

```bash linux-build.sh
# Build release x64 (default)
./linux-build.sh

# Build debug x64 with 8 cores
./linux-build.sh --arch=x64 --config=debug --cores=8

# Build for ARM64
./linux-build.sh --arch=arm64
```

## Install Runtime Data

After a successful build, run the data-installation script to copy runtime data files to the output directory:

```bash
./linux-install-data.sh
```

<Note>
  The Linux build produces the **MTA:SA dedicated server only**. The game client (with FMOD audio features) runs on Windows.
</Note>
