Midnight Purple SA has grown from a minimal FMOD integration into a full-featured spatial audio system covering DSP effect chains, per-vehicle sound simulation, raycasted occlusion, and server-driven playback sync. The sections below catalogue every feature that has shipped in the current build, grouped by subsystem, followed by the planned work ahead.

<Note>
  All listed features in the Completed section are fully implemented in the current build. The version label in-game reads **Midnight Purple 1.7** (upstream base synced 2026-05-16).
</Note>

## Completed Features

### Core Audio

- тЬЕ **Doppler effect** тАФ `fmodSetSoundVelocity` sets a channel's velocity vector so FMOD calculates pitch shift automatically based on relative motion between source and listener
- тЬЕ **Pause / resume** тАФ channels can be individually paused and resumed via `fmodSetChannelPaused`
- тЬЕ **Streaming** тАФ `fmodCreateStream` for music and long-form audio with low RAM overhead
- тЬЕ **Loop state toggle** тАФ loop mode can be enabled or disabled on a live channel without restarting playback
- тЬЕ **Read-back functions** тАФ `fmodGetSoundVolume`, `fmodGetSoundPitch`, and equivalent getters for all major channel parameters
- тЬЕ **Per-resource sound tracking** тАФ sounds are automatically freed when the owning resource stops, with no manual cleanup required
- тЬЕ **FMOD error propagation** тАФ all `fmod*` functions return `false` on failure; `fmodGetLastError()` returns the FMOD error code and description
- тЬЕ **`fmodGetVersion()`** тАФ returns the linked FMOD library version string at runtime

### DSP & Effects

- тЬЕ **Additional DSP types** тАФ low-pass, high-pass, flanger, chorus, and distortion effect nodes available via `fmodAddChannelEffect`
- тЬЕ **DSP chain query** тАФ `fmodGetChannelEffects` returns the ordered list of DSP nodes currently applied to a channel
- тЬЕ **Volume categories** тАФ SFX, Ambient, and Music sub-groups under the Master ChannelGroup; per-category volume control via `fmodSetCategoryVolume`

### Network & Sync

- тЬЕ **Server-triggered playback** тАФ the bundled `fmod_sync` resource allows server-side scripts to trigger FMOD sounds on clients using a client-side cache; no raw file paths are sent over the network

### Spatial Audio

- тЬЕ **Occlusion / obstruction** тАФ `fmodSetChannelOcclusion` applies geometry-based attenuation and muffling to a channel, accepting direct and reverb occlusion factors
- тЬЕ **Air-absorption LPF** тАФ a distance-proportional low-pass filter is applied automatically to 3D sounds beyond 20 m, simulating atmospheric high-frequency rolloff

### Vehicle Audio

- тЬЕ **Engine sound layer** тАФ the `hikari_Engine` sound is routed through FMOD, enabling pitch and DSP manipulation tied to RPM
- тЬЕ **Per-vehicle sound bank** тАФ `engine_table.lua` maps vehicle model IDs to custom FMOD sound assets, allowing fully unique engine audio per vehicle
- тЬЕ **Rev limiter / gear shift events** тАФ BOV (blow-off valve) and backfire / anti-lag system (ALS) sounds are triggered on gear change and rev-limiter hit events
- тЬЕ **Exhaust 3D positioning** тАФ the exhaust sound channel's 3D position is updated each frame to match the vehicle's exhaust bone in world space
- тЬЕ **Tunnel echo & reverb** тАФ a raycast-based detection system applies tunnel reverb presets when the vehicle is inside a tunnel or underpass
- тЬЕ **Rev limiter distortion** тАФ a DSP distortion node is blended in on the engine channel as RPM approaches and holds the rev limiter
- тЬЕ **Per-vehicle occlusion** тАФ a throttled line-of-sight raycast (fired at most once every 200 ms per vehicle) drives `fmodSetChannelOcclusion` for other players' vehicles, muffling sound through solid geometry

### Miscellaneous

- тЬЕ **Version label** тАФ in-game version string updated to **Midnight Purple 1.7**

## Planned Features

The following items are under active consideration or development for future releases. Nothing listed here is guaranteed or committed to a specific version.

### Core Audio

- ЁЯФ▓ **`fmodSetChannelPitch`** тАФ expose per-channel pitch control as a dedicated Lua function independent of Doppler
- ЁЯФ▓ **Seek / playback position** тАФ `fmodSetChannelPosition` and `fmodGetChannelPosition` for scrubbing within a stream or sample
- ЁЯФ▓ **Fade helpers** тАФ timed fade-in and fade-out utilities built on top of the existing volume API

### DSP & Effects

- ЁЯФ▓ **Reverb zones** тАФ trigger a reverb preset on all channels within a defined world-space sphere, driven by `fmodSetReverbProperties`
- ЁЯФ▓ **DSP parameter control** тАФ expose individual parameter knobs (e.g. chorus depth, LPF cutoff) via `fmodSetEffectParameter`

### Spatial Audio

- ЁЯФ▓ **Geometry occlusion API** тАФ register static mesh faces with FMOD's geometry engine so occlusion is computed natively rather than through raycasts
- ЁЯФ▓ **Early reflections** тАФ per-room early reflection settings for indoor environments (warehouses, tunnels) without requiring a full reverb preset

### Vehicle Audio

- ЁЯФ▓ **Tyre / road-surface sounds** тАФ material-aware tyre noise routed through a dedicated FMOD channel with per-surface DSP presets
- ЁЯФ▓ **Wind noise channel** тАФ speed-proportional wind audio with high-frequency roll-off at low speeds

### Network & Sync

- ЁЯФ▓ **Reliable playback sync** тАФ sequence numbers and client-side drift correction for `fmod_sync` to keep long ambient loops aligned across players
